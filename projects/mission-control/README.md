# Mission Control

## Project Overview
CLAW's central dashboard and command center. A Next.js web application that provides real-time visibility into all workspace operations, projects, agents, memory, automation, and system health.

## Stack
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + custom globals
- **Port:** 3040

## Running
```bash
npm run dev
```

## Features
- System-wide health dashboard
- Multi-project overview with channel mappings
- Agent & identity viewer
- Memory & daily journal browser
- Cron job & automation monitor
- Live activity feed
- Command reference / cheat sheet
- Tools & capabilities inventory
- Settings & configuration viewer
